# CuPlus – Product Overview

## Purpose
CuPlus is a Personal Knowledge Management (PKM) app that helps users capture, organize, and protect their ideas, projects, notes, and daily habits in one place. It runs as a cross-platform app (web + Android via Capacitor).

## Key Features

- **Projects** – Rich-text documents with categories, thumbnails, word count tracking, archive, and recycle bin (soft-delete lifecycle).
- **Notes** – Quick-capture notes with grid view and individual note editing.
- **Habits** – Daily habit tracker with goals, progress logging, increment/toggle/count measure types, and 14/30-day history.
- **Sanctum** – Password-protected encrypted space (personal & official) using AES-GCM + PBKDF2 client-side encryption; server never sees plaintext.
- **Tasks** – Task management with upcoming task widget on the home dashboard.
- **Scratchpad** – Lightweight scratch area for ephemeral notes.
- **Quick Capture** – Fast input for capturing thoughts without navigating away.
- **Remember This** – Sidebar memory widget for pinned reminders.
- **Archive & Recycle Bin** – Non-destructive content lifecycle (active → archived → removed → permanently deleted).
- **Theme** – Light/dark theme with flash-free initialization via inline script.
- **Profile** – Account management, appearance settings, contact, data export, storage info, and account deletion.

## Target Users
Individuals who want a unified personal workspace for writing, planning, habit-building, and storing sensitive information securely.

## Value Proposition
Combines a rich-text editor, habit tracker, encrypted vault, and note-taking in a single offline-capable PWA/Android app backed by Firebase.
